#Metadata Extraction API
##Running the application

To rum the application it's necessary to perform the 3 following commands form the core directory.

* export FLASK_APP=Metadata_Extraction_API

* export FLASK_ENV=development

* flask run

The project also contain a very simple HTML page to help test the upload function the other must be tested as normal.

##Functionality 

The application is build using Flask Factory design which provides more modelability and efficiency in large projects.
The application also counts with automated system that generate unique keys for the files upon extraction and organise 
in specific folders to facilitate the search reduce the processing power demanded in of a necessary retrieval.
